编辑栏快捷键
渲染代码: F5 或 SHIFT+Enter
保存到缓存: CTRL + S
从缓存加载: CTRL + L
清除缓存: CTRL + SHIFT + \
下载代码: CTRL + SHIFT + S
重置视图: CTRL + Enter
3D视图 鼠标控制
旋转XZ轴: 左键
旋转XY轴: 右键 或 ALT + 左键
平   移: 中键 或 SHIFT + 左键
缩   放: 滚轮 或 CTRL + 左键
支持的文件格式
jscad: OpenJSCAD (导入/输出)
scad: OpenSCAD (导入)
stl: STL format (导入/输出)
amf: AMF format (导入/输出)
dxf: DXF format (导入/输出)
x3d: X3D format (输出)
svg: SVG format (导入/输出)
